# Leaflet Marker Images

Place the following Leaflet default marker images in this directory:

- marker-icon.png
- marker-icon-2x.png
- marker-shadow.png

You can download these from: https://github.com/leaflet/leaflet/tree/master/dist/images

Or use these direct links:
- https://raw.githubusercontent.com/leaflet/leaflet/main/dist/images/marker-icon.png
- https://raw.githubusercontent.com/leaflet/leaflet/main/dist/images/marker-icon-2x.png
- https://raw.githubusercontent.com/leaflet/leaflet/main/dist/images/marker-shadow.png
